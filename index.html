Что такое ЯзРП?

ЯзРП (Язык Русского Программирования) — это встроенный скриптовый язык для Unity, вдохновлённый C#, но использующий русскую лексику. Позволяет создавать объекты, управлять физикой, UI, мультиплеером и игровой логикой без написания обычных C# скриптов. Идеально для обучения, прототипирования и русскоязычного геймдева.

Читаемый русский синтаксис

Команды: Написать на "панель", Напечатать Квадрат, создать объект, Создать текст — всё понятно с первого взгляда.

🎮

Локальный мультиплеер

Создайте двух игроков за секунды: Создать игрока "Красный" управление "ad" и "Синий" управление "стрелки". Каждый собирает свои монетки.

🧩

Геометрические фигуры + UI

Команды создать объект ... форма "Куб" цвет "зелёный" и Создать текст ... создают игровой мир и интерфейс прямо из скрипта.

💾

Сохранение скриптов

Встроенный редактор с кнопками "Сохранить", "Загрузить" + команды Путь сохранения "persistent" и Сохранить скрипт.

🧪 Как протестировать прямо сейчас?

  1. Создайте новый 3D проект в Unity (версия 2021.3 LTS или новее).
  2. Импортируйте скачанный .unitypackage (все скрипты ЯзРП).
  3. Откройте сцену GameScene.unity (там есть Canvas, InputField и GameManager).
  4. Нажмите Play → введите команды или загрузите демо-скрипт → нажмите «Выполнить».
  5. Управляйте персонажами и наблюдайте за созданием объектов!

⭐ Совет: В поле ввода уже будет пример мультиплеерной игры, можно менять координаты монеток и добавлять своих врагов.
